Definition du concept

Tw-Dash est une application reservée uniquement aux administrateurs de Twendeno, Sa mission principale est de créer le réseau routier par departement.

Legende
  • Line

    Elle herite de geometrie, une geometrie est un ensemble de coordonnée géographique. Elle peut etre disposé sous forme de polygone, de point , de linestring (suite de point) etc.
    Exemple:
    Un arret de bus est une geometrie de type point,
    Une direction Grand marché - Aeroport est une geometrie de type linestring (suite de point). Les champs obligatoires pour créer une ligne sont:

    • Name : le nom de la ligne
    • Departement: Il s'agit du departement dans lequel la ligne sera crée
    • Town: Il s'agit de la ville dans laquelle la ligne sera crée
    • Geometry type: Il s'agit du type de geometrie qu'heritra la ligne
  • Station

    Elle herite de coordonnée, une station est une coordonnée géographique.
    Les champs obligatoires pour créer une stations sont:

    • Name : le nom de la station
    • Latitude et longitude : coordonnées géographiques
  • Direction

    Elle permet de lier une ligne à une station.
    Les champs obligatoires pour créer une direction sont:

    • Name : le nom de la direction
    • Departure et Arrival : determine le debut et fin de ligne (le terminus)
    • geometry : est l'identifiant de la ligne qui sera lier à la station
    • station : est l'identifiant de la station qui sera lier à la geometrie
